# Katkıda Bulunma Rehberi

Nebevi Rehber projesine katkıda bulunmak istediğiniz için teşekkür ederiz! Bu proje, topluluğun katılımlarıyla büyümeyi ve gelişmeyi hedefler.

## Nasıl Katkıda Bulunabilirsiniz?

1.  **Hataları Bildirin:** Karşılaştığınız sorunları veya hataları GitHub "Issues" sekmesinden bize bildirin.
2.  **Özellik İsteyin:** Projeye eklenmesini istediğiniz özellikleri veya içerikleri önerin.
3.  **İçerik Geliştirin:** Mevcut içerikleri zenginleştirin veya yeni rehberler ekleyin.
4.  **Yazım Hatalarını Düzeltin:** Gördüğünüz yazım veya anlatım bozukluklarını düzeltin.

## Katkı Süreci (Pull Request)

1.  Bu repoyu "Fork" edin.
2.  Kendi bilgisayarınıza klonlayın (`git clone ...`).
3.  Yeni bir dal (branch) oluşturun (`git checkout -b yeni-ozellik`).
4.  Değişikliklerinizi yapın ve kaydedin (`git commit -m 'Yeni özellik eklendi'`).
5.  Fork ettiğiniz repoya gönderin (`git push origin yeni-ozellik`).
6.  GitHub üzerinden bir "Pull Request" (Çekme İsteği) oluşturun.

## İçerik Standartları

*   İçerikler anlaşılır ve sade bir dille yazılmalıdır.
*   Kaynak belirtilmesi gereken yerlerde referans verilmelidir.
*   Markdown formatına uygun düzenlemeler yapılmalıdır.

Katkılarınız için şimdiden teşekkürler!
